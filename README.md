# Scrap in Python

## Run Program
- Example : `python Scrap.py out/namefile.txt -p 100`
![example](./doc/example.PNG)
- More info : `python Scrap.py -h`
![help](./doc/help.PNG)

## Requirement
- python3 (recommended)
